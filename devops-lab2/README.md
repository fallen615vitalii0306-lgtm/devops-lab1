# Лабораторная работа №2 по DevOps

## Часть 1
Развертывание веб-сервиса в Kubernetes с использованием:
- ConfigMap
- Deployment
- Service

## Часть 2
Создание Helm chart и обновление приложения с помощью команды `helm upgrade`

## Структура проекта
- `k8s/app.yaml` - манифест Kubernetes
- `helm/my-app/` - Helm chart
- `report/` - отчет по лабораторной работе

## Результат
Сервис развернут, обновление выполнено через Helm.
